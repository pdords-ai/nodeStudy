## 📚 학습 목표

<!-- 
이 프로젝트를 통해 달성할 수 있는 학습 목표들입니다.

- React의 기본 개념과 문법 이해 (React의 기본 원리 파악)
- Hooks 사용법 숙지 (useState, useEffect 등 핵심 훅들)
- 컴포넌트 간 통신 방법 학습 (부모-자식, 형제 컴포넌트 간 데이터 전달)
- 실제 프로젝트에서 사용되는 패턴들 익히기 (실무에서 바로 사용할 수 있는 기술들)
- Spring Boot 백엔드 서버 구축 및 연동 (풀스택 개발 경험)
- 데이터베이스 연동 및 CRUD 작업 (데이터 영속성 이해)
- 실시간 통신 구현 (WebSocket을 통한 양방향 통신)

## 🛠️ 설치 및 실행 (초보자용 상세 가이드)

<!-- 
이 프로젝트를 처음 시작하는 분들을 위한 아주 자세한 가이드입니다.

### 📋 준비물 (필요한 것들)
- **컴퓨터**: Windows, Mac, Linux 중 아무거나
- **인터넷 연결**: 프로그램을 다운로드하기 위해 필요해요
- **시간**: 약 30분 정도

### 🔧 1단계: Node.js 설치하기

**Node.js가 뭔가요?**
- 웹사이트를 만드는 도구예요 (마치 요리할 때 필요한 주방 도구와 같아요)
- JavaScript라는 언어로 프로그램을 만들 수 있게 해줘요

**설치 방법:**
1. **웹브라우저**를 열어주세요 (Chrome, Firefox, Safari 등 아무거나)
2. 주소창에 `https://nodejs.org`를 입력하고 엔터를 눌러주세요
3. **"LTS"**라고 써진 초록색 버튼을 클릭해주세요 (가장 안정적인 버전이에요)
4. 다운로드된 파일을 실행하고 "Next" 버튼을 계속 눌러주세요
5. 설치가 끝나면 컴퓨터를 다시 시작해주세요

**설치 확인하기:**
1. **명령 프롬프트**를 열어주세요
   - Windows: `Windows 키 + R` → `cmd` 입력 → 엔터
   - Mac: `Command + Space` → `터미널` 입력 → 엔터
2. `node --version`을 입력하고 엔터를 눌러주세요
3. 숫자가 나오면 성공! (예: v18.17.0)

### 📁 2단계: 프로젝트 폴더 열기

**방법 1: 명령 프롬프트 사용**
1. 명령 프롬프트를 열어주세요
2. `cd C:\react`를 입력하고 엔터를 눌러주세요 (이 폴더로 이동)
3. `dir`을 입력하면 폴더 안의 파일들이 보여요

**방법 2: 파일 탐색기 사용**
1. 파일 탐색기를 열어주세요
2. C 드라이브 → react 폴더를 찾아서 열어주세요

### 📦 3단계: 필요한 프로그램들 다운로드하기

**이 단계에서 뭘 하는 거예요?**
- React를 사용하기 위해 필요한 작은 프로그램들을 다운로드해요
- 마치 게임을 설치할 때 필요한 파일들을 다운로드하는 것과 같아요

**어떻게 하나요?**
1. 명령 프롬프트에서 `C:\react` 폴더에 있는지 확인해주세요
2. `npm install`을 입력하고 엔터를 눌러주세요
3. 조금 기다리면 다운로드가 완료돼요 (2-3분 정도 걸려요)

**성공하면 이런 메시지가 나와요:**
```
added 1234 packages in 2m
```

### 🚀 4단계: 웹사이트 실행하기

**이제 진짜 웹사이트를 만들어볼 시간이에요!**

1. 명령 프롬프트에서 `npm start`를 입력하고 엔터를 눌러주세요
2. 잠깐 기다리면 자동으로 웹브라우저가 열려요
3. `http://localhost:3000`이라는 주소로 웹사이트가 열려요

**성공하면 이런 화면이 보여요:**
- 🚀 React 학습 프로젝트라는 제목
- 여러 가지 예제 버튼들
- 각 버튼을 클릭하면 다른 기능을 볼 수 있어요

### 🎉 5단계: 웹사이트 사용하기

**이제 마음껏 실험해보세요!**
- 각 예제 버튼을 클릭해보세요
- 숫자를 바꿔보세요
- 텍스트를 입력해보세요
- 코드를 수정해보세요

**코드를 수정하면:**
- 웹사이트가 자동으로 새로고침돼요
- 변경사항이 바로 화면에 나타나요

### 🛑 웹사이트 종료하기

**웹사이트를 그만 보고 싶을 때:**
1. 명령 프롬프트에서 `Ctrl + C`를 눌러주세요
2. "Y"를 입력하고 엔터를 눌러주세요
3. 웹사이트가 종료돼요

### ❓ 문제가 생겼을 때

**"npm이 인식되지 않는다"는 오류가 나와요:**
- Node.js가 제대로 설치되지 않았어요
- 1단계부터 다시 해보세요

**"포트가 이미 사용 중"이라는 오류가 나와요:**
- 다른 프로그램이 3000번 포트를 사용하고 있어요
- 명령 프롬프트를 모두 닫고 다시 `npm start`를 해보세요

**웹사이트가 안 열려요:**
- 인터넷 연결을 확인해주세요
- 방화벽 설정을 확인해주세요
- 컴퓨터를 다시 시작해보세요

### 🎯 다음 단계

**웹사이트가 잘 실행되면:**
- 각 예제를 하나씩 클릭해보세요
- 코드를 수정해보세요
- 새로운 기능을 추가해보세요

### 📋 추가 준비물
- **Java**: Spring Boot 서버를 실행하기 위해 필요해요
- **시간**: 약 15분 정도

### ☕ 1단계: Java 설치하기

**Java가 뭔가요?**
- Spring Boot 서버를 실행하기 위한 도구예요
- 마치 자동차를 움직이기 위한 엔진과 같아요

**설치 방법:**
1. **웹브라우저**를 열어주세요
2. 주소창에 `https://adoptium.net`를 입력하고 엔터를 눌러주세요
3. **"Eclipse Temurin 17 (LTS)"**를 선택해주세요
4. **"Windows x64"** (또는 Mac이면 "macOS x64")를 선택해주세요
5. 다운로드된 파일을 실행하고 "Next" 버튼을 계속 눌러주세요
6. 설치가 끝나면 컴퓨터를 다시 시작해주세요

**설치 확인하기:**
1. 명령 프롬프트를 열어주세요
2. `java --version`을 입력하고 엔터를 눌러주세요
3. "openjdk 17"이라는 글자가 나오면 성공!

### 🗂️ 2단계: Spring Boot 서버 실행하기

**이제 진짜 서버를 실행해볼 시간이에요!**

**Windows 사용자:**
1. 명령 프롬프트를 열어주세요
2. `cd C:\react\backend`를 입력하고 엔터를 눌러주세요
3. `start-server.bat`을 입력하고 엔터를 눌러주세요

**Mac/Linux 사용자:**
1. 터미널을 열어주세요
2. `cd /path/to/react/backend`를 입력하고 엔터를 눌러주세요
3. `chmod +x start-server.sh`를 입력하고 엔터를 눌러주세요
4. `./start-server.sh`를 입력하고 엔터를 눌러주세요

**Gradle Wrapper가 없는 경우:**
1. `gradle wrapper`를 입력하고 엔터를 눌러주세요
2. `./gradlew bootRun`을 입력하고 엔터를 눌러주세요

**성공하면 이런 메시지가 나와요:**
```
🚀 React Learning Backend Server가 시작되었습니다!
📡 서버 주소: http://localhost:8080
🗄️ H2 데이터베이스 콘솔: http://localhost:8080/h2-console
```

### 🌐 3단계: 서버 확인하기

**서버가 잘 실행되었는지 확인해보세요:**

1. **웹브라우저**를 열어주세요
2. 주소창에 `http://localhost:8080`을 입력하고 엔터를 눌러주세요
3. "Whitelabel Error Page"라는 페이지가 나오면 성공! (이건 정상이에요)

**데이터베이스 콘솔 확인:**
1. 주소창에 `http://localhost:8080/h2-console`을 입력하고 엔터를 눌러주세요
2. "Connect" 버튼을 클릭해주세요
3. 데이터베이스 테이블들이 보이면 성공!

### 🔗 4단계: React와 Spring Boot 연결하기

### 1. useState 훅 예제 (숫자 세기 배우기)
**이 예제에서 배우는 것:**
- **기본 카운터**: + 버튼을 누르면 숫자가 커지고, - 버튼을 누르면 숫자가 작아져요
- **사용자 정보**: 이름, 나이, 이메일 같은 정보를 저장해요
- **할 일 목록**: 해야 할 일을 추가하고 삭제할 수 있어요
- **입력 폼**: 텍스트를 입력하면 화면에 나타나요

**왜 중요한가요?**
- React에서 가장 기본이 되는 기능이에요
- 마치 요리할 때 가장 기본이 되는 밥 짓기와 같아요

**어떻게 사용하나요?**
1. "useState 훅 예제" 버튼을 클릭해주세요
2. + 버튼을 눌러보세요 (숫자가 커져요!)
3. - 버튼을 눌러보세요 (숫자가 작아져요!)
4. 텍스트를 입력해보세요

**학습 포인트 (고급자용):**
- `useState` 훅의 기본 사용법
- 객체와 배열 상태 업데이트 방법
- 함수형 업데이트 패턴

### 2. useEffect 훅 예제 (자동으로 실행되는 기능)
**이 예제에서 배우는 것:**
- **자동 실행**: 페이지가 열리면 자동으로 실행되는 기능
- **조건부 실행**: 특정 조건이 맞을 때만 실행되는 기능
- **정리 기능**: 사용하지 않는 것을 정리하는 기능
- **데이터 가져오기**: 인터넷에서 정보를 가져오는 기능

**왜 중요한가요?**
- 웹사이트가 똑똑하게 동작하게 해줘요
- 마치 자동차가 자동으로 시동이 걸리는 것과 같아요

**어떻게 사용하나요?**
1. "useEffect 훅 예제" 버튼을 클릭해주세요
2. 페이지가 열리면 자동으로 실행되는 것을 확인해보세요
3. 버튼을 눌러서 다른 기능도 테스트해보세요

**학습 포인트 (고급자용):**
- `useEffect`의 다양한 사용 패턴
- 의존성 배열의 중요성
- 클린업 함수로 메모리 누수 방지

### 3. Props 전달 예제 (데이터 주고받기)
**이 예제에서 배우는 것:**
- **기본 데이터**: 텍스트, 숫자, 참/거짓 같은 간단한 데이터 전달
- **복잡한 데이터**: 여러 정보가 들어있는 데이터 전달
- **함수 전달**: 버튼을 누르면 실행되는 기능 전달
- **내용 포함**: 컴포넌트 안에 다른 내용 넣기

**왜 중요한가요?**
- 컴포넌트들이 서로 정보를 주고받을 수 있어요
- 마치 편지를 주고받는 것과 같아요

**어떻게 사용하나요?**
1. "Props 전달 예제" 버튼을 클릭해주세요
2. 각 버튼을 눌러서 다른 데이터가 어떻게 전달되는지 확인해보세요
3. 텍스트를 입력해보세요

**학습 포인트 (고급자용):**
- Props의 다양한 타입과 사용법
- Props 검증과 기본값 설정
- 컴포넌트 재사용성 향상

### 4. 조건부 렌더링 예제 (조건에 따라 보여주기)
**이 예제에서 배우는 것:**
- **조건부 표시**: 특정 조건에 따라 다른 내용을 보여줘요
- **테마 변경**: 밝은 테마와 어두운 테마를 바꿀 수 있어요
- **버튼 상태**: 버튼을 누르면 내용이 바뀌어요

**왜 중요한가요?**
- 웹사이트가 상황에 따라 다르게 보여요
- 마치 날씨에 따라 옷을 바꿔 입는 것과 같아요

**어떻게 사용하나요?**
1. "조건부 렌더링 예제" 버튼을 클릭해주세요
2. 테마 버튼을 눌러서 색깔이 바뀌는 것을 확인해보세요
3. 다른 버튼들도 눌러보세요
